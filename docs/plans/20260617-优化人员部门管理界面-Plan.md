# 优化人员/部门/职能管理界面滚动 — 实施计划

**Goal:** 修复人员管理、部门管理、职能管理三个页面的滚动条行为：页面内容超出时在当前内容区域内部滚动，而非整体页面滚动。确保人员管理页面左右两侧高度一致。

**Architecture:** 纯前端 CSS 改动。核心思路是让每个页面的根容器 `height: 100%` + `overflow: hidden`，再让内部内容区 `flex: 1` + `overflow: auto` 实现内部滚动。不改动业务逻辑、不改后端。

**Tech Stack:** Vue 3 + Element Plus + CSS (scoped)

## 全局约束

- 只改 CSS，不改 script 逻辑
- 遵循 Element Plus 组件现有结构，不添加多余 wrapper
- 改动最小化，只加必要的 height / overflow / flex 属性
