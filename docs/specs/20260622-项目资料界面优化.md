# 项目资料界面滚动优化

> 日期：2026-06-22　|　状态：已完成

## 1. 为什么要做（背景）

项目编辑/查看页面中，多个 tab 没有合理的高度约束。内容超出视口时触发页面级滚动，导致：
- 任务列表的横向滚动条被推到页面最底部，必须滚到底才能操作
- 上下滚动时 tab 头部、页面标题跟着一起滚出视口
- 操作日志出现页面级和内部级两条纵向滚动条
- 各 tab 底部有空白区域，未填满可用空间

## 2. 要达成什么（需求）

- [x] 项目编辑/查看页面高度锁定在视口内，不再产生页面级滚动
- [x] 任务列表 tab：横向 + 纵向滚动条均在可视区域内，分组按钮和列头固定
- [x] 操作日志 tab：只有一条内部纵向滚动条，筛选栏固定
- [x] 文件资料 tab：表格填满可用空间、内部滚动，表头和底部按钮固定
- [x] 甘特图 tab：flex 填满 tab 区，不再用硬编码 calc 高度
- [x] 基本信息 tab：表单区内部滚动，底部保存按钮固定
- [x] 成员列表 tab：表格内部滚动，工具栏和底部按钮固定
- [x] 任务计划 tab：表格填满可用空间、表头固定内部滚动
- [x] 基本信息、成员列表底部只保留保存按钮，去掉取消/返回按钮

## 3. 怎么算做完（验收标准）

- [x] 打开任意项目编辑/查看页，页面本身不可纵向滚动
- [x] 切到「任务列表」tab，横向滚动条在可视区域内可见，上下滚动列内容时列头不动
- [x] 切到「操作日志」tab，只有一条纵向滚动条，筛选栏不随滚动消失
- [x] 切到「文件资料」tab，表格填满、内部滚动，表头/底部按钮固定
- [x] 切到「甘特图」tab，flex 填满可用空间，布局不错位
- [x] 基本信息 tab 表单区内部滚动，保存按钮固定底部
- [x] 成员列表 tab 表格内部滚动，工具栏/保存按钮固定
- [x] 任务计划 tab 表格填满、内部滚动，工具栏固定
- [x] 所有改动不影响只读查看模式

## 4. 实现摘要

### 4.1 页面根节点约束（ProjectEditor.vue）

```css
.project-editor {
  height: calc(100vh - 56px);   /* 视口 - 顶部导航栏 */
  overflow: hidden;              /* 切断页面级滚动 */
  display: flex;
  flex-direction: column;        /* 纵向 flex 分配 */
}
```

`.page-header` 加 `flex-shrink: 0` 固定标题头。

### 4.2 Tabs 弹性布局（ProjectEditor.vue）

```css
.editor-tabs { flex: 1; min-height: 0; display: flex; flex-direction: column; }
.editor-tabs :deep(.el-tabs__header) { flex-shrink: 0; }           /* tab 头固定 */
.editor-tabs :deep(.el-tabs__content) { flex: 1; min-height: 0; }  /* 撑满剩余 */
.editor-tabs :deep(.el-tab-pane) { flex: 1; min-height: 0; overflow-y: auto; }  /* 默认内部滚动 */
.editor-tabs :deep(.tab-pane-fill) { overflow: hidden; }           /* fill 型 tab 接管滚动 */
```

需要定制滚动的 tab 加 `class="tab-pane-fill"` 并在内部容器上控制滚动。所有 fill 型 tab 统一模式：

```
tab-pane-fill (overflow:hidden)
  └─ wrapper (height:100%; flex column; overflow:hidden)
      ├─ toolbar (flex-shrink:0)            ← 固定
      ├─ scroll-area (flex:1; min-height:0)  ← 精确填满
      └─ footer (flex-shrink:0)             ← 固定（可选）
```

### 4.3 任务列表 / 看板（ProjectEditor.vue）

`el-tab-pane` 加 `class="tab-pane-fill"`。模板外层加 `<div class="board-tab-wrapper">` 包裹。

```css
.board-tab-wrapper { height: 100%; display: flex; flex-direction: column; overflow: hidden; }
.board-container { flex: 1; overflow: auto; min-height: 0; }        /* 双向滚动在此 */
.board-column { display: flex; flex-direction: column; }            /* 列内纵向 flex */
.board-column-header { flex-shrink: 0; }                            /* 列头固定 */
.board-column-body { flex: 1; min-height: 0; overflow-y: auto; }    /* 卡片区纵向滚 */
```

### 4.4 操作日志（ProjectOperationLogTab.vue）

- `el-tab-pane` 加 `class="tab-pane-fill"`
- `el-card` 加 `class="oplog-card"` → `height:100%; flex column`
- 时间线外包裹 `<div class="oplog-scroll">` → `flex:1; overflow-y:auto`

```css
.oplog-card { height: 100%; display: flex; flex-direction: column; }
.oplog-card :deep(.el-card__body) { flex: 1; min-height: 0; display: flex; flex-direction: column; overflow: hidden; }
.oplog-scroll { flex: 1; overflow-y: auto; min-height: 0; }
```

### 4.5 文件资料（ProjectFileTab.vue）

- 父级 tab pane 加 `class="tab-pane-fill"`
- `.project-file-tab` → `height:100%; flex column; overflow:hidden`
- `.file-toolbar` / `.form-footer` → `flex-shrink:0` 固定
- 表格外包裹 `<div class="file-table-wrap">` → `flex:1; min-height:0; overflow:hidden`
- `el-table` → `height="100%"` 表格自身填满并内部滚动

```css
.project-file-tab { height: 100%; display: flex; flex-direction: column; overflow: hidden; }
.file-toolbar { flex-shrink: 0; }
.file-table-wrap { flex: 1; min-height: 0; overflow: hidden; }
.form-footer { flex-shrink: 0; }
```

### 4.6 基本信息（ProjectEditor.vue）

`el-tab-pane` 加 `class="tab-pane-fill"`，外层加 `<div class="basic-tab-wrapper">`。

```css
.basic-tab-wrapper { height: 100%; display: flex; flex-direction: column; overflow: hidden; }
```

`el-form` 加 `style="flex:1;min-height:0;overflow-y:auto"`，底部 `.form-footer` 只保留保存按钮。

```
basic-tab-wrapper
  ├─ el-form (flex:1; overflow-y:auto)  ← 表单内部滚动
  └─ form-footer (flex-shrink:0)        ← 保存按钮固定
```

### 4.7 成员列表（ProjectEditor.vue）

`el-tab-pane` 加 `class="tab-pane-fill"`。去掉 `el-card`，改用 toolbar + table-wrap 结构。

```css
.member-tab-wrapper { height: 100%; display: flex; flex-direction: column; overflow: hidden; }
.member-toolbar { display: flex; justify-content: space-between; align-items: center; margin-bottom: 12px; flex-shrink: 0; }
.member-table-wrap { flex: 1; min-height: 0; overflow-y: auto; }
```

底部 `.form-footer` 只保留保存按钮，去掉取消/返回按钮。

```
member-tab-wrapper
  ├─ member-toolbar (flex-shrink:0)       ← 标题 + 操作按钮固定
  ├─ member-table-wrap (flex:1; overflow-y:auto)  ← 表格内部滚动
  └─ form-footer (flex-shrink:0)          ← 保存按钮固定
```

### 4.8 任务计划（ProjectEditor.vue）

`el-tab-pane` 加 `class="tab-pane-fill"`。去掉 `el-card`，改用 toolbar + table-wrap 结构。

```css
.tasks-tab-wrapper { height: 100%; display: flex; flex-direction: column; overflow: hidden; }
.tasks-toolbar { margin-bottom: 12px; flex-shrink: 0; }
.tasks-table-wrap { flex: 1; min-height: 0; overflow: hidden; }
```

`el-table` 改 `height="calc(100vh - 350px)"` → `height="100%"`，表头固定内部滚动。

```
tasks-tab-wrapper
  ├─ tasks-toolbar (flex-shrink:0)        ← 标题 + 操作按钮固定
  └─ tasks-table-wrap (flex:1; overflow:hidden)
      └─ el-table height="100%"           ← 表格内部滚动
```

### 4.9 甘特图（ProjectEditor.vue）

`el-tab-pane` 加 `class="tab-pane-fill"`。

```css
.gantt-wrapper { height: 100%; display: flex; flex-direction: column; }
.gantt-toolbar { flex-shrink: 0; }
.gantt-container { flex: 1; min-height: 0; }   /* 替代 calc(100vh-330px) */
```

### 4.10 附带改动

- 操作列的 `fixed='right'` 改为 CSS `sticky` 定位（`.col-sticky-right`），避免 Element Plus 树表格固定列层级错位
- 任务列表看板使用 `taskStatusLabel` / `priorityLabel` 工具函数显示状态/优先级标签
- 基本信息底部只保留「保存」按钮，去掉「取消/返回」
- 成员列表底部只保留「保存」按钮，去掉「取消/返回」

## 5. 影响范围

| 文件 | 改动类型 |
|------|---------|
| `ProjectEditor.vue` | 页面根节点 + tabs 弹性布局 + 基本信息/成员列表/任务计划/任务列表/甘特图 fill 模式 |
| `ProjectOperationLogTab.vue` | 加 `tab-pane-fill` + 滚动容器 + 样式新增 |
| `ProjectFileTab.vue` | 加 `tab-pane-fill`（父级）+ flex 填充 + `height="100%"` 表格 |

所有改动仅限于 CSS 布局和少量模板结构调整，不涉及业务逻辑变更。
